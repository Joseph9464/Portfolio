Classification des Prénoms Togolais

Ce projet consiste à déterminer si un prénom est togolais ou non à l'aide d'un modèle de machine learning.

Description

Ce projet vise à développer un modèle capable de classifier les prénoms togolais en utilisant un dataset d'exemples. L'objectif est d'offrir une solution simple pour identifier l'origine des prénoms dans un contexte sociétal.

Technologies Utilisées

  • Python
  • Scikit-Learn
  • Pandas
  • Jupyter

Résultats

Le modèle a obtenu une précision de 85% en testant sur un échantillon de 1000 prénoms. La performance peut être améliorée en optimisant les hyperparamètres.
